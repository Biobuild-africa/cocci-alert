What happens behind the scenes

Every interaction follows the same five-step controlled pipeline. The farmer never knows about steps 3-5 — they just receive their triage advice.

1

Farmer input

14 bounded questions answered via USSD menu or WhatsApp buttons. No free-text PII captured.

▸
2

Rule engine

Deterministic Python engine; Emmanuel's 12-sign decision tree; no ML in the clinical path.

▸
3

Triage result

Emergency / Act now / Keep monitoring + paraphrased action message in farmer's language.

▸
4

Anonymized log

Phone number hashed with rotating salt. Geographic resolution capped at LGA. Consent flag persisted.

▸
5

FCI4Africa data layer

Aggregated LGA-month rollups pushed to Food Systems Dashboard. Only consenting records included.

This is a controlled prototype demonstration produced as TRL 4 evidence for the FCI4Africa OC1 proposal. It uses synthetic data. No real farmer data has been collected. Live USSD/WhatsApp deployment is a Phase 2 deliverable. BioBuild Africa Ltd · RC 9604822.
